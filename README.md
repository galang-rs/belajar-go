# Latihan Pemrograman

Repositori ini berisi dokumentasi dan hasil pengerjaan latihan coding yang saya lakukan.

## Penjelasan Proyek

Dalam pengerjaan latihan di dalam repositori ini, saya menggunakan pendekatan kolaborasi bersama Artificial Intelligence (AI) dengan pembagian tugas sebagai berikut:

- 🤖 **Pembuatan Soal:** Kasus atau masalah yang harus dipecahkan dibuat secara otomatis menggunakan AI.
- 🧪 **Unit Testing:** Seluruh *unit test* di-generate oleh AI untuk memastikan spesifikasi soal terpenuhi.
- 🧑‍💻 **Penyelesaian Logika (Solusi):** Saya murni mengerjakan logika pemrogramannya sendiri. Pengerjaan kodenya dibatasi *hanya di dalam scope fungsi (func) yang sudah disediakan*.

## Tujuan

Pendekatan ini digunakan untuk mengasah kemampuan *problem-solving*, melatih penulisan logika kode yang benar, dan membiasakan diri membuat program yang bisa lolos dari skenario pengujian (*Test-Driven Development*).
